## 🧰 الحزمة التقنية (إلزامية)

### الخادم (Backend)
| المكوّن | التقنية | الغرض |
|--------|---------|-------|
| Runtime/Framework | **Node.js + Express.js** | الخادم وواجهة الـ API |
| المحرّك اللحظي | **Socket.io** | مزامنة الماسحات لمنع تكرار مسح الـ QR |
| معالجة الصور | **sharp** | دمج الاسم/QR على تصميم الدعوة |
| توليد QR | **qrcode** | إنشاء رموز QR قياسية قابلة للمسح |
| الأمان | **helmet + express-rate-limit** | الحماية، خلف Cloudflare WAF |
| الإرسال | **Unifonic (واتساب/SMS)** + **nodemailer/SendGrid (إيميل)** | عبر فئات خدمة مجرّدة (Abstract Service Classes) |

### قاعدة البيانات
| المكوّن | التقنية |
|--------|---------|
| القاعدة | **MongoDB** (بيئة إنتاج) |
| الـ ODM | **Mongoose** |
| المتطلب | مخططات تدعم استعلامات تحليلية سريعة (تجميع الحضور/الغياب لحظيًا) |

### التخزين السحابي
- **AWS S3** أو متوافق (DigitalOcean Spaces) أو **Cloudinary**.
- الاستخدام: تخزين التصاميم المرفوعة + صور الدعوات النهائية قبل الإرسال.

### تطبيق الجوال (المالك + المنسّق)
| المكوّن | التقنية |
|--------|---------|
| الإطار | **React Native + Expo** |
| الكاميرا/الماسح | **expo-camera** أو **react-native-vision-camera** (محسّن للمسح المتواصل في الإضاءة الضعيفة) |
| التنقّل | **react-navigation** |
| الحالة | **React Context** أو **Zustand** (خفيف) |
| الشبكة/الكاش | **TanStack Query (React Query)** لجلب وتخزين بيانات اللوحة |

### مواقع الويب (الدعوة + التعريفي)
| المكوّن | التقنية |
|--------|---------|
| الإطار | **Next.js** (متجاوب، بحجم شاشة الجوال) |
| التنسيق | **Tailwind CSS** + دعم RTL |

> **قاعدة التنفيذ:** الالتزام الصارم بهذه الحزمة. لا اقتراح بدائل لقاعدة البيانات
> (لا SQLite/PostgreSQL) ولا لبيئة الخادم.

---

## 🔐 الأمان والأداء
- **helmet** لرؤوس HTTP الآمنة.
- **express-rate-limit** على نقاط الدخول الحسّاسة (المصادقة، المسح، رفع الملفات).
- وضع الخادم خلف **Cloudflare WAF** للحماية من DDoS.
- مصادقة المنسّق برقم الجوال + OTP (بدون كلمات مرور) → JWT قصير العمر.
- روابط الدعوات بمعرّفات غير قابلة للتخمين (UUID/توقيع).
- توقيع/تشفير حمولة الـ QR لمنع التزوير.
